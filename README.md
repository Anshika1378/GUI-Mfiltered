# MFILTERED Dashboard

Responsive Next.js dashboard for Ad Fraud detection and User Identity verification.

## Features

- Next.js App Router with TypeScript
- Tailwind CSS styling
- shadcn-style UI components
- Theme toggle with `next-themes`
- Responsive layout for mobile, tablet, and desktop
- Separate modules under `/ad-fraud` and `/user-identity`
- Basic test coverage with Vitest and React Testing Library

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Run tests:
   ```bash
   npm test
   ```

## Deployment

Deploy to Vercel or your preferred host using the `next build` command.

## Notes

This repository is scaffolded with the required app structure, theme switching, responsive UI, and module navigation as requested.
